本次作业中完成版为dictionary.html,测试版为smalldict.html
使用说明如下：
1.当用户没有或没有全部输入就运行时会有弹窗：the input can't be null
2.当用户输入错误的单词时，会有弹窗：not all of them are words
3.当用户输入正确的单词但不等长时，会有弹窗：their lengths are not the same
4.当两个单词无路径时会显示：sorry！ There is no path
5.当用户输入正确时，会在按钮下方显示路径。
完成版说明：
完成版可以直接使用，但运行时间一般较长，正常运行约十秒左右，具体时间与视情况而定。
测试版说明：
测试版供内部测试用，在关键地方用alert()显示信息，同时也是开发过程中进行单元测试的文件，
其字典规模较小，方便测试。
